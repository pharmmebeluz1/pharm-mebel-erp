MEBEL360° — GITHUB VA RENDERGA YUKLASH
======================================

1. Ushbu ZIP faylni kompyuterda oching.
2. ZIP ichidagi barcha fayl va papkalarni belgilang.
3. GitHubdagi pharm-mebel-erp loyihasining asosiy joyiga yuklang.
4. Izohga quyidagini yozing:

Mebel360° dinamik bosqichlar va ishonchli mijoz tizimi qo‘shildi

5. Commit changes tugmasini bosing.
6. Render → Manual Deploy → Deploy latest commit ni bosing.
7. Deploy live chiqqach saytni Ctrl + F5 bilan yangilang.

MUHIM:
- Render Diskni o‘chirmang.
- Ma’lumotlar bazasi faylini o‘chirmang.
- Eski buyurtmalar saqlanadi; yangi ustunlar avtomatik qo‘shiladi.
